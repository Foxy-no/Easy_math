# What is a function $f(x)$?

## Simple idea

$x$ -> [ ] -> $y$

You can think of it as a box. You can input a value ($x$) in the box, and it outputs another value ($y$)

---

<details>
  <summary>Box Examples</summary>
  
  ### Example 1: $f(x) = x - 1$
  
  2 -> [ 2 - 1 ] -> 1
  
  1 -> [ 1 - 1 ] -> 0
  
  You can write this same example as:
  
  $f(2) = 2 - 1 = 1$
  
  $f(1) = 1 - 1 = 0$
  
  ### Example 2: $f(x) = \frac{x - 1}{2^x}$
  
  $f(2) = \frac{2 - 1}{2^2} = \frac{1}{4}$
  
  $f(1) = \frac{1-1}{2^1} = 0$
  
  $f(0) = \frac{0-1}{2^0} = -1$
  
</details>

---

This idea of function as a box is also useful to understand something about functions. 1 input, 1 output. I mean that for each value you can give to $x$, there must be only one output value for $y$, otherwise what you have, your box, is not to be considered a function.
