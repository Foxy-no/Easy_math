### Domain

- Definition: The domain of a function (usually written as an interval) represents all possible values for $x$ that return a real value for $f(x)$.

- Basically: if for some specific values of x the function does not exist in R, the domain is not all R but R - {all those values of x}.

**How you actually do it**: relevant situations
1. ^n√g(x) with n even & n € N
2. g(x) as divisor
3. log(g(x)) with base n € R & n > 0
